Introduction to CSS

CSS stands for Cascading Style Sheet, is a styling Language. A styling language is a one which is used to design a Web-page and add responsiveness to the Web-page. In recent times CSS has become much popular for Web-designing.

Why do we need CSS?

CSS is the programming language of the INTERNET. You see CSS (JS for short) all over the Internet.

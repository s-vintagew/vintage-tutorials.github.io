Lists

These are the bulleted texts that are displayed in Web-page. There are two types of lists in HTML:- Ordered and Unordered and Descriptive.

As the name suggests, Ordered List refer to those which are numbered (either in 1-2-3 or a-b-c or i-ii-iii) whereas unordered lists have no such numbering (consists of circle- ○, disc- ● , square- ■ and blank also) and Descriptive refer to the list with a certain head/title and then below it the description.

For creating ordered list we use the <ol> tag and for creating unordered list we use the <ul> tag and for creating descriptive list we use the <dl> tag. The attributes are listed below.

  • <li> tag:- List Item tag used to list an item/entity inside the specified list.
  • <dt> tag:- This specifies the text as the title of the Descriptive List.
  • <dd> tag:- This specifies the text as the description of the title of the Descriptive List.
  • type:- For ordered list types are: 1,a,i,A,I; For unordered list types are: disc, circle, square & none
  • start:- Specifies the starting of the numbered list.
  • reverse:- Commands the numbered list to be in reverse order.

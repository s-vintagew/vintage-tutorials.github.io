Header

It is the part of the Code that is displayed above the browser. It basically contains every information about the code, the tags for indexing, the media ports and the title that has to be displayed in the particular Tab of the Browser. <head> tag is used to define this section of the HTML Code.

The following elements can go inside <head> element

  • <title>
  • <meta>
  • <script>
  • <noscript>
  • <link>
  • <base>
  • <style>
    •


Title

This tag is used within <head> tag and it displays the title of the Web-page in that particular Tab of the Browser.


Meta

This tag is used within <head> tag and it stores all the meta-data such as view, width, title, description, designation etc about the Website/Web-page.


Script

This tag is used within <head> tag and it marks the start an end of the Scripting Language inside HTML Code. Inside it we write the codes for the front-end/back-end calculation that are required to be performed in the Web-page.


noScript

This tag is used within <head> tag and it defines an alternate content to be displayed to users that have disabled scripts in their browser or have a browser that doesn't support script.


Link

This tag is used within <head> tag and tag defines the relationship between the current document and an external resource. Often used to link to external style sheets, this element is an empty element, it contains attributes only.


Base

This tag is used within <head> tag and it specifies the base URL and/or target for all relative URLs in a document.


Style

This tag is used within <head> tag and it encloses all the codes written as a part of styling language which helps to design and decorate the Web-page.

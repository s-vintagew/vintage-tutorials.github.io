Image

Images make the website attractive in cases. There are background images that we have already discussed in the Body section. What comes next is to set a standalone image. In HTML we can do this task by using the <img> tag.

The tag is followed by some attributes as listed below:-

  • src: This specifies the path of the image that has to be displayed. It could be an https:// link or a direct image from the local file system.
  • width: Specifies the width of the Image.
  • Height: Specifies the height of the Image.
  • alt: This contains text within it which will be displayed in case there is a error in loading the Image.

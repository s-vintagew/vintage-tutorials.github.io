Tables

In HTML Tables play a major role to organize files and texts and show them in a well-fashioned output

The Elements and Attributes of <table> tag are:

  • <thead>: This is used when we have to mention a particular section of the table as the heading section. This part is always displayed on the top of the table.
  • <tbody>: This is used when we have to mention a particular section of the table as the body section. This part is always displayed in between thead and tfoot of the table table.
  • <tfoot>: This is used when we have to mention a particular section of the table as the footer section. This part is always displayed on the bottom of the table.
  • <tr>: Used to declare starting of the rows in a table.
  • <td>: Used to declare a particular column inside a row which makes a cell and there data can be written.
  • <th>: Similar to td tag with a difference that this tag highlights the text marking the presence of some heading text.
  • <colgroup>: Used to format columns in the table.
  • <caption>: Used to add caption to the table.

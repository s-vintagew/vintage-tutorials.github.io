CSS Tables

The look of an HTML Table can be improved greatly by the use of CSS. The properties are:

  • table-borders
  • table-size
  • table-alignment
  • table-style

table-border

To specify table borders in CSS, use the border property. The example below specifies a black border for <table>, <th>, and <td> elements:

Example 
	table, th, td {
  		border: 1px solid black;
	}
Output
Firstname Lastname
Peter Griffin
Lois Griffin

table-size

The width and height properties can be used to set the size of a table. Value can be in % or px.

Example 
	table {
  		width: 100%;
	}

	th {
  		height: 70px;
	}

table-alignment

The text-align property sets the horizontal alignment (like left, right, or center) of the content in <th> or <td>.
By default, the content of <th> elements are center-aligned and the content of <td> elements are left-aligned.
To center-align the content of <td> elements as well, use text-align: center:

Example 
	td {
  		text-align: center;
	}

table-style

Table can be styles using:

  • background-color: To color the background
  • padding: To give space in between margin to text
  • border-style: To specify which border style has to be used. It could be dashes, double, groove, outset, inset, dotted, solid.
  • border-color: To give color to the border
  • border-radius etc.

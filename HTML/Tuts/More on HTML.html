Structure of HTML

Basic Structure 
    <!DOCTYPE html>
    <html>
    <head>
        <title>Page Title</title>
    </head>
    <body>

    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>

    </body>
    </html>

Attributes

These are the codes/commands or statements that associate a particular element/tag in doing it's task the way we desire.
For example, if we want to align a certain text to the right then along with the <p> tag we use align="right" which aligns the text in the paragraph as right alignment

Elements

Tags or Elements are the most important part of the HTML Coding. They comprise of all the instructions we give in the form of codes.

HTML can be split into two parts. One part is display in the browser part that is the Header part done using the <head> tag and the main part of the browser that is the body part which is written inside <body> tag.

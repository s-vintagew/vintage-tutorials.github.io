Classes or Containers

tag defines a division or a section in an HTML document. The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The <div>tag is easily styled by using the class or id attribute.

It has mainly two attributes

  • Class: Defines a name for that particular selected section.
  • Id: Defines a id/name for that particular selected section.



Marquee

Along with these HTML also has a provision for constant text animation one that we often see in advertisement etc. This is known as <marquee> tag. A description of this is shown below.

This is an example of marquee tag

It has the listed attributes

  • hspace: This specifies the space of the text from the Margin in the Horizontal Direction.
  • vspace: This specifies the space of the text from the Margin in the Vertical Direction.
  • bgcolor: Adds back-ground color to the banner/element.
  • behavior: This tag tell the text to go either left to right/right to left/ alternate.

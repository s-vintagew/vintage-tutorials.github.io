HTML is the standard markup language for Web pages.
With HTML you can create your own Website.
HTML is easy to learn - You will enjoy it!



Consisting of various elements and their attributes along with styles (CSS which is in the later part of the tutorial) and some real time events makes it fun for developing websites.

But unlike other Programming Languages HTML doesn't have a fixed compiler. It totally depends on the browser for parsing depending on which HTML Version it supports. Now this compiler factor is very annoying at times because you won't even know that you made a mistake somewhere in the code unless you see the output and this becomes hectic when the Code is huge enough. But there's thrill in that I must say.

There are 3 languages which a Web-Developer must learn

  1. HTML: for defining the content/structure of the Web-page
  2. CSS: for specifying the layout/design of Web-page
  3. JavaScript: for programming the behavior of the Web-page

To begin with Web Designing we learn HTML which provides the layout of the Web-page or you can say Website(a collection of web-pages). This is the foundation/blue-print/base of any particular website no matter how good and attractive and appealing it looks.

HTML stand for Hyper Text Markup Language, and as said earlier provides the basic structure of any web-page.

After HTML we have CSS which stands for Cascading Style Sheet which is used for providing designs and styles to the Web-page. And following this we have Java-script that is used for the back-end calculations etc.

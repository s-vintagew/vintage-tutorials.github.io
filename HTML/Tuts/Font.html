Font

This is the most useful tag when it comes to designing using HTML Only. Font tag can be used to specify font style/type, font-color, font-size etc. Unfortunately this tag is no longer supported in the higher versions of HTML (>4).

The attributes of <font> tag are:-

  • font-face:- Use to set various font types to our Codes.
  • color:- Assigns color to the Texts. Can be assigned using Color Name (BLUE) or Hex Code #6600ff or RGB values (0, 153, 0).
  • size:- Specifies the size in px(pixels) or normal(1-7range) or in em (1em=16px)

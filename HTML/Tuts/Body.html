Body

This is the most important part of the HTML Code. This contains all the elements, statements, content of the Web-page. The below listed are the attributes of the <body> tag

  • background
  • bgcolor
  • link
  • alink
  • vlink
  • text
    •

background

This attribute of Body tag is used to define a certain image path and set the image as the background of the Body.


bgcolor

This attribute of Body tag is used to define a color for the Web-page Body.


link

This attribute of Body tag is used to set color for all the hyper-links that are mentioned in the body. Default color of the links are Blue.


alink

This attribute of Body tag is used to set color for all the hyper-links that are active at that instant in that Browser. Default color of the links are Red.


vlink

This attribute of Body tag is used to set color for all the hyper-links that have been visited already in the browser. Default color of the links are Violet.


text

This attribute of Body tag is used to set color for all the texts inside the body element. It is kind of a universal color throughout the Web-page.

Introduction to JavaScript

JavaScript is a Scripting Language. A scripting language is a one which doesn't need a compiler step and are rather interpreted/parsed. In recent times JavaScript has become much popular for front-end and back-end calculation in web developing.

JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web. JavaScript is easy to learn.

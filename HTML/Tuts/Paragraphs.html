Paragraphs

This tag is used within body tag to group a certain number of sentences into a single paragraph. Normally to separate paragraphs we have to break the lines using <br> tag (used for breaking the line or taking the cursor to the next line) but <p> tag itself separates different paragraphs with proper spacing.

Demo 
        <!DOCTYPE html>
<html>
<body>

<p>This is a paragraph.</p>
<p>This is a paragraph.</p>
<p>This is a paragraph.</p>

</body>
</html>

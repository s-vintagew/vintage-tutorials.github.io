<B>
Description: - This tag is used to BOLD the texts so that the specified texts within it are highlighted. This tag needs to be closed.

<I>
Description: - This tag is used to ITALIC or slant the texts so that the specified texts within it are highlighted. This tag needs to be closed.

<U>
Description: - This tag is used to Underline the texts so that the specified texts within it are highlighted. This tag needs to be closed.

<big>
Description: - This tag increases the text size by 2px and so the specified texts within this tag looks a little bigger. This tag needs to be closed.

<small>
Description: - This tag decreases the text size by 2px and so the specified texts within this tag looks a little smaller. This tag needs to be closed.

<em>
Description: - This tag is used when we have to emphasize on the text and this tag makes it italic. This tag is similar to tag. This tag needs to be closed. This tag is often used in places when in between statements we have to mark certain words which while reading we need to emphasize on.

<strong>
Description: - This tag is used to make a text look bold. This tag is similar to <b> tag. This tag needs to be closed. Often used in places where in between statements certain words are to be made strongly focused.

<sub>
Description: - This tag is used in places where we need to subscript certain texts. This tag is not an open tag and thus, it needs to be closed. For example H2O

<sup>
Description: - This tag is used in places where we need to superscript certain texts. This tag is not an open tag and thus, it needs to be closed. For example X2

<ins>
Description: - In certain places we need to keep the previous data shown and also display the new data. For example, while discounts in online shopping the previous price and the current price both are shown like this 230 200 Rs. In these cases, this tag can be useful. This tag needs to be closed.

<del>
Description: - In certain places we need to strike out the previous data and display the new data. For example, while discounts in online shopping the previous price is strike off and the current price both are shown like this 230 200 Rs. In these cases, this tag can be useful. This tag needs to be closed.

<bdo>
Description: - The full form of these tag is Bi-Directional Override. This tag is used to write some text from left to right or vice versa. This is not an open tag and thus, needs to be closed. For example HTML is showed Left to Right but it can be shown as HTML which reads left to right.

<pre> 
Description: - This tag represents preformatted text. 
Whatever is specified within this tag is displayed exactly as written including the blank spaces. For example this paragraph itself.
This tag has to be closed.


<samp>
Description: - To display certain text in monospace (computerized/typewriter font) to make it look different than the other texts. This tag needs to be closed.

<code>
Description: - This tag is used in places where we want to bind up certain code from a programming language. This tag needs to be closed.

<kbd>
Description: - This tag is used in places where we have to print the name of the keyboard keys. For Example ctrl. This tag needs to be closed.

<font>
Description: - This tag is used to edit the font of the text. The size, color and the family can be changed using this font. This tag needs to be closed. In this webpage different types of font faces and colors are used.

<var>
